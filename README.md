# To-Do-List
A user friendly to do list app with a soft and calming pastel color palette graphical user interface. 
## Main Page:
![image](https://user-images.githubusercontent.com/63769772/112407219-47669900-8d48-11eb-8ee6-63b278abe363.png)

Here, users can create a new to do list, or load an existing to do list. 
Loading existing to do lists will enable users to make changes to items in the to do list.

## Create New To Do List:
![image](https://user-images.githubusercontent.com/63769772/112407386-94e30600-8d48-11eb-9e76-4a73bcc159d3.png)

When a new to do list is created, users can simply click to choose a date from the calendar and add, remove, edit, delete, pin, or unpin tasks.

![image](https://user-images.githubusercontent.com/63769772/112407930-7a5d5c80-8d49-11eb-9dc2-e7fb7ef728fb.png)

This app supports drag and drop, which means that the tasks on the left can be rearranged according to the user's preference by simply dragging and dropping them.

## Pin tasks:
![image](https://user-images.githubusercontent.com/63769772/112407551-e095af80-8d48-11eb-804b-cbd451c175ef.png)

After selecting a task, users can pin the task where the task that is pinned will move to the top of the list and a star will 
be appended to that task.

To unpin, simply select the task and click the unpin button. The star will disappear.

## Check Box:
![image](https://user-images.githubusercontent.com/63769772/112407713-294d6880-8d49-11eb-821f-5364e0b0d17f.png)

After completing a task, users can click on the check box in front of the task to place a check mark to indicate that the task has been completed.

## Clear Task:
![image](https://user-images.githubusercontent.com/63769772/112408108-c90af680-8d49-11eb-9f2e-4b0ef23e1233.png)

A button that should be used with caution as it will clear the entire task list.

## Save Your List!
![image](https://user-images.githubusercontent.com/63769772/112408138-d7591280-8d49-11eb-827f-f9b10e9f3a1d.png)

After adding tasks to the list, users should click 'save' to save their lists so that it could be loaded, viewed, and edited in the future.
